O que é CSS?

Cascate Style Sheet (folha de estilos em cascata)

Cascade Style Sheet são folhas de estilo que formatam o HTML para a exibição em navegadores da web, tanto desktop quanto mobile.

Imagine o HTML como uma régua de estêncil: ele tem algumas formas, mas a disposição dos elementos, as cores, isso quem decide é o css.

Calc:

				calc([valor A] [operação] [valor N])

Suporte desde o ie9.

				--nome-var: [VALOR];

FLex box

A web é dinâmica demais para caixas

  • Desde 2009, última atualização 2017
  • O elemento pai dita as regras para os filhos. Por padrão, transforma em colunas.
  • Todos os filhos diretos são afetados (na dúvida, mude a estrutura)

Grid model

Chegou a hora da web ser bi-dimensional.

  • Lançado em maio/2017
  • Elemento bi-direcional: trabalha com X e Y
  • O elemento pai dita as regras para os filhos
  • Organize todos os elementos a partir de um elemento-pai
  • Indicado para a página toda.
